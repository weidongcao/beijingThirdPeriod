# 管综项目说明

----------
## 使用说明 ##
 - 部署项目时根据情况看document目录下的文档

## 压缩包文件说明 ##

 - lib文件夹
项目依赖的jar包

 - config文件夹
项目的配置文件,单独查看用,应用执行的时候与这个没有关系

 - document文件夹
项目说明文档

 - BeiJingThirdPeriod.jar文件
应用jar包

 - oracle_history_export.sh脚本
跑历史任务

 - oracle_real_time_export.sh脚本
跑实时任务,从Oracle取数据

 - real_time_import_bcp.sh脚本
跑实时任务，从Bcp文件取数据
